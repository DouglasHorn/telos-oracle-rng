# Telos Ping

## Setup

To begin, navigate to the project directory: `telos-ping/`

    mkdir build && mkdir build/ping

    chmod +x build.sh

    chmod +x deploy.sh

## Build

    ./build.sh ping

## Deploy

    ./deploy.sh ping { account-name } { mainnet | testnet | local }
